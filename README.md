# Dotfiles
Dotfiles used for the i3 configuration on Ubuntu.

## Files

i3 folder: Contains the config file that specifies looks, shortcuts and behaviour <br />
Xresources: Modifies the URxvt terminal

## Software Used

Linux Distribution: Ubuntu <br />
Terminal: URxvt

## Aditional Software
 - i3lock (optional)
 - Rofi
 - Ranger
