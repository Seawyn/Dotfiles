# Dotfiles
Dotfiles used for the I3 configuration on Ubuntu.

## Files

i3 folder: Contains the config file that specifies looks, shortcuts and behaviour
Xresources: Modifies the URxvt terminal

## Software Used

Linux Distribution: Ubuntu Mate <br />
Terminal: URxvt <br />
